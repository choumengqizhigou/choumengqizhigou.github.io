# 开发工具Visual Studio 2022 安装与激活

`这篇主要是图文演示安装 Visual Studio 2022 ，如果已经安装过可直接跳过。`

可以看到代码咱们用 **记事本** 也可以去写！如果你有这么操作过，你会发现有诸多的不利，回到代码本身，那么多字符错一个，可能程序就无法 `编译` ，一些细小的错误也不容易被发现，写错一个大小写，你也不知道，等到 `编译` 的时候，提示你哪一行有问题，你再回来仔细找到错误然后修改代码，再去 `编译` 。。。不说代码错误的问题，就是代码格式，你也要手动去把控，不然挤在一块看起来也不清晰。

那么我们尝试抛开代码的问题，回到 `编译` 的这个操作流程，又要指定代码文件，又要指定输出的文件，其实还有一些编译参数咱们都还没开始填，而且我们目前还只有一个代码文件，如果有多个代码文件，就光指定代码文件地址，都够你难受一番的。

很显然上述的这些问题，是让人极其难受的，这时候我们就需要开发一个 **新的记事本** 程序，让这个程序能够更加便捷的帮我们开发程序，比如什么 `代码提示` 、 `代码文本格式化` 、 `代码书写错误提示` 、 `管理代码文件` 、 `一键编译代码并运行` 。。。

这就该轮到我们的 **集成开发环境（IDE）** 登场了，这个 **新的记事本** 程序就是 `Visual Studio 2022` 后简称 `VS 2022` 。


## 下载

原先的 `VS` 是支持离线安装的，下载一个离线包，直接安装就可以，不过现在呢，需要下一个 `Install` 的小文件，然后选择你要安装的一些组件，就开始边下载，边安装了！（**需要全程联网**）

咱们就开始去下载吧！

首先访问我们的 `VS 2022` [官方网站](https://visualstudio.microsoft.com/zh-hans/vs/) 。然后移动到下载按钮，在弹出的版本选择界面中，选择 `Professional 2022` 然后点击就会开始下载了。

网址：https://visualstudio.microsoft.com/zh-hans/vs/

下载下来的是一个 `VisualStudioSetup.exe` ，直接双击打开就会进入安装界面了。

## 开发工具激活

因为我们下载的是 Professional 版本，此版本是一个收费版本，所以我们可以尝试激活此版本，如果你现在不激活也可以，或者你可以下载 Community 版本，这个是社区版，免费使用的，咱们后续的操作无关版本，用什么版本操作流程都一样。

首先我们需要一个激活码：`TD244-P4NB7-YQ6XK-Y8MMM-YWV2J` 

![20230517163454](https://blog-image-group.oss-cn-shanghai.aliyuncs.com/BlogImages/20230517163454.png)

然后启动 `VS 2022` ，`点击 继续但无需代码(W)`

然后在新界面的上方依次选择 `帮助` -> `Register Visual Studio` ，并且点击 `Register Visual Studio` 

![20230518075226](https://blog-image-group.oss-cn-shanghai.aliyuncs.com/BlogImages/20230518075226.png)

在弹出的界面点击 `使用产品密钥解锁` 

![20230518075404](https://blog-image-group.oss-cn-shanghai.aliyuncs.com/BlogImages/20230518075404.png)

输入前面说的产品密钥 `TD244-P4NB7-YQ6XK-Y8MMM-YWV2J` 点击 `应用` 等待激活

![20230518075520](https://blog-image-group.oss-cn-shanghai.aliyuncs.com/BlogImages/20230518075520.png)

然后就可以看到激活成功的提示了！
