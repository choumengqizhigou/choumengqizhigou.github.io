# 1、C#+SQLServer-项目介绍

## 项目介绍

你好，我是丑萌气质狗。

此项目使用 `C#` 配合 `SQLServer数据库` 开发小案例。帮助理解数据库软件的概念，了解SQL语句，熟悉操作流程。

同时对 `ADO.NET` 和ORM框架 `Entity Framework 6` 进行对比演示说明。

## 开发环境

**开发工具（IDE）：** `Visual Studio 2022`

**开发框架：** `Console控制台应用程序` 、 `WinForm`

**.NET 平台：** `.NET Framework 4.7.2`

**数据库：** `SQL Server 2022`

**ORM框架：** `Entity Framework 6.5.2`

**注：下载项目如果无法连接数据库，记得找到 `MyDBContext.cs` 文件，修改里面的连接字符串（MyDBContext.cs中有配置说明）。**
