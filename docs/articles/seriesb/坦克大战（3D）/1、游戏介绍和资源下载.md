# 1、介绍与资源下载

## 游戏介绍

双人坦克游戏，两个玩家可以分别操作两辆坦克进行炮弹射击，直至一名玩家死亡。

## 开发以及资源说明

本次使用的编辑器版本是 `Unity 2022.3 LTS` 长期支持版，本次内容使用的版本是 `Unity 2022.3.3` ，目前官方可能已经更新到 `Unity 2022.3.xxx` ，大家如果想下载我的这个版本，可以去 [Unity官网](https://unity.cn/releases/lts) 上找到此版本，然后点击 `从Unity Hub下载` 即可。

**为了保持一致性的操作，非常建议大家使用一样的版本，会减少很多不必要的麻烦。**

另外补充一句，这是 `Unity` 在国内的最后一个长期支持版本，，后续的 `Unity6` 已经不对国内支持，国内目前拥有自己的版本，也就是我们所说的 `团结引擎` ，它是在 `Unity 2022.3 LTS` 版本上演化而来，后续会添加很多具有本土特色的功能。
